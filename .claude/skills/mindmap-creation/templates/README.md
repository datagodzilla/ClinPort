# Mind Map Templates

This folder contains templates for creating different types of mind maps.

## Available Templates

| Template | Type | Best For |
|----------|------|----------|
| `concept-mindmap-template.md` | Concept | Understanding topic structure and relationships |
| `process-mindmap-template.md` | Process | Visualizing workflows and procedures |
| `domain-mindmap-template.md` | Domain | Mapping entire knowledge areas or fields |
| `problem-solution-mindmap-template.md` | Problem-Solution | Analyzing problems and mapping solutions |
| `comparative-mindmap-template.md` | Comparative | Comparing and contrasting concepts |

## When to Use Each Template

### Concept Mind Map
- Learning complex subjects
- Knowledge organization
- Understanding hierarchies

### Process Mind Map
- Documenting workflows
- Planning implementations
- Understanding sequences

### Domain Mind Map
- Curriculum design
- Field exploration
- Comprehensive understanding

### Problem-Solution Mind Map
- Troubleshooting
- Strategic planning
- Root cause analysis

### Comparative Mind Map
- Decision making
- Understanding trade-offs
- Feature comparisons

## Usage

1. Copy the appropriate template
2. Replace placeholders with your content
3. Add/remove branches as needed
4. Save to `docs/[topic]/mindmaps/[type]/`
